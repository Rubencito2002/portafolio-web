Sobre mí

En 2024, terminé el Grado Superior en Desarrollo de Aplicaciones Web, en el IES Hermanos Machado. Durante mis prácticas en FISEVI, dentro del Hospital Virgen del Rocío pero más concretamente en el Instituto de Biomedicina de Sevilla (IBIS), adquirí una valiosa experiencia como Desarrollador de Aplicaciones.

Mi enfoque principal es el frontend, donde estoy aprendiendo conocimientos en ReactJS y Angular, y tengo sólidos conocimientos en HTML, CSS, Bootstrap y Javascript.

En backend, he utilizado principalmente Python, con el framework web Django y DRF para desarrollar mis APIs REST, y también he realizado proyectos utilizando Java. Tengo experiencia con varias bases de datos relacionales, en especial MySQL.

Y en cuanto a despliegues, tengo conocimientos en varios servicios de Amazon Web Services (AWS) como buckets S3 para páginas estáticas y archivos, instancias EC2 para proyectos Django que necesitan ejecutar código y bases de datos RDS, además de GitHub Pages para proyectos estáticos cuando el proyecto no necesita otros servicios de AWS.
